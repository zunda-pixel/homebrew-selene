## Selene Homebrew

Selene Homebrew Self Hosting

https://github.com/zunda-pixel/selene

### Install

```shell
brew tap zunda-pixel/selene
brew install zunda-pixel/selene/selene
```

### Uninstall

```shell
brew uninstall zunda-pixel/selene/selene
```

#### Option

```shell
brew untap zunda-pixel/selene
```
